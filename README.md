# 📊 Social Media Analytics + AI Dashboard

An end-to-end data analytics and machine learning project built with Python.

## 🚀 Features

- FastAPI backend for data processing
- CSV file upload and analysis
- Automatic data cleaning
- Engagement score prediction model
- Streamlit dashboard for visualization
- Real-time analytics API

## 🧠 Tech Stack

- Python
- FastAPI
- Pandas
- Scikit-learn
- Streamlit

## 📂 Project Structure

- app.py → Backend API (FastAPI)
- dashboard.py → Frontend dashboard (Streamlit)
- requirements.txt → Dependencies

## 📊 ML Logic

Engagement score is calculated using:

likes * 0.4 + shares * 0.3 + comments * 0.2 + views * 0.1
